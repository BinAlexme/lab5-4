Создайте функцию, которая принимает аргументы «текст» и «имя файла».

Функция должна:

1. Открыть файл (если файл не существует, создать его).
2. Добавить текст в конец файла с новой строки.
3. Вывести на экран содержимое всех четных строк файла.

Используйте встроенную функцию `open()` с режимом `'a'` для добавления текста в файл. Для чтения файла используйте режим `'r'`.

Для работы с четными строками используйте срезы или проверку индексов.
Для получения всех строк файла можно использовать `f.readlines()`. Чтение данных из файла реализовать с
помощью контекстного менеджера. Для получения четных строк можно использовать функцию `enumerate()`.

#### Пример работы (вывода) / записи в файл:

```
example.txt

Первая строка
Вторая строка
Третья строка
Четвертая строка
```

```
append_to_file("Пятая строка", "example.txt")
```

```
example.txt

Первая строка
Вторая строка
Третья строка
Четвертая строка
Пятая строка

```
