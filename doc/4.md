Напишите функцию вычисления площади прямоугольника, аргументами для которой будут длина и ширина.

Напишите для этой функции декоратор, который будет выводить информацию о том,
что функция была вызвана с такими-то позиционными и именованными аргументами (`args`, `kwargs`), а также ее имя.

Декоратор должен принимать вашу функцию и возвращать ее результат. Для получения имени функции используйте атрибут `__name__`.
Этот атрибут позволяет получить имя функции, класса или модуля.

#### Пример работы (вывода):

```
Функция calculate_area вызвана с аргументами:
Позиционные аргументы: (5, 10)
Именованные аргументы: {}
Площадь прямоугольника: 50
```
